# 📊 Superstore Sales Dashboard

## Project Overview
This project is an interactive Power BI Sales Dashboard built using the Superstore dataset.

## Tools Used
- Power BI
- DAX
- CSV Dataset

## Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Profit Margin KPI
- Sales by Month
- Sales by Category
- Sales by Region
- Sales by Segment
- Interactive Slicers (Year, Region, Category)

## Skills Demonstrated
- Data Cleaning
- Data Visualization
- DAX Measures
- Dashboard Design
- Business Insights

Sample Superstore Dataset
